# Set up

## SDK

### [SDKMAN](https://sdkman.io)

#### [installation](https://sdkman.io/install/)

설치
```shell
# for bash
curl -s "https://get.sdkman.io" | bash
# for zsh
curl -s "https://get.sdkman.io" | zsh
```

터미널 사용 초기화
```shell
source "$HOME/.sdkman/bin/sdkman-init.sh"
```

설치 확인
```shell
sdk version
```

```log

native: 0.7.34 (macos aarch64)
```

jdk 설치
- **애플 실리콘 칩을 이용하는 M 시리즈 맥인 경우 x86 빌드의 JDK가 아닌 aarch64로 빌드된 JDK를 받아서 설치하도록 한다**

```shell
sdk install java 21.0.6-librca
```

> install java 뒤에서 Tab 키를 누르면 identifier를 가능한 범위까지 자동완성 해준다.  
> Tab 을 두번 누르면 설치 가능한 목록을 보여준다.  

디폴트 jdk
- 터미널에 들어갔을 때 기본으로 적용되는 JDK(Java) 버전을 default로 지정해둘 수 있다.

```shell
sdk default java 21.0.5-librca
```

프로젝트별 jdk 버전 적용

1. JDK를 설치하고
2. 프로젝트 루트 폴더에서 sdk use로 사용할 JDK를 선택하고
    ```shell
    sdk use java 21.0.5-librca
    ```
3. sdk env init으로 초기화하고 선택한 JDK 정보를 저장
    ```shell
    sdk env init
    ```
   선택한 JDK 버전 정보는 .sdkmanrc 파일에 저장
4. 이후에 `sdk env` 명령으로 저장된 버전 정보에 맞는 JDK 활성화
5. 저장된 JDK 버전이 설치되어 있지 않으면 다음 명령으로 설치
    ```shell
    sdk env install
    ```

> 프로젝트 폴더에 들어갈 때 자동으로 저장된 JDK 버전이 활성화 되게 만들려면  
> ~/.sdkman/etc/config의 다음 줄을 수정  
> sdkman_auto_env=true

## HTTP Client

### [HTTPie](https://httpie.io)

#### [CLI Installation](https://httpie.io/docs/cli/installation)

설치
```shell
# Install httpie
brew update
brew install httpie
```

```shell
# Upgrade httpie
brew update
brew upgrade httpie
```
